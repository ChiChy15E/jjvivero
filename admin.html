⚙️ Configuración de Imágenes

Haz clic para alternar entre imágenes externas (URLs) y locales (archivos)

📋 Instrucciones:

🌐 Imágenes Externas (URLs)
  • Usa URLs de Unsplash, tu servidor, etc.
  • Cambios inmediatos sin subir archivos
  • Ideal para pruebas rápidas
  • Requiere conexión a internet
📁 Imágenes Locales (Archivos)
  • Sube archivos a la carpeta 'images/'
  • Mayor control y velocidad
  • Funciona sin internet
  • Nombres automáticos generados
💡 Recomendaciones de Tamaño:
Hero/Carrusel: 1200x800 píxeles (3:2) | Productos: 400x300 píxeles (4:3) | Formatos: JPG, PNG, WebP
